# AI Personalized Study Assistant

An AI-powered study assistant built using Streamlit and Ollama.

## Features

- AI Study Notes Generator
- AI Quiz Generator
- Flashcards Generator
- AI Chatbot
- Voice Input Support
- PDF Notes Download
- Notes History
- Self Evaluation Quiz

## Technologies Used

- Python
- Streamlit
- Ollama
- TinyLlama
- SpeechRecognition
- FPDF

## How to Run

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Ollama

```bash
ollama run tinyllama
```

### Start Streamlit App

```bash
streamlit run app.py
```
